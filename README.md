# web2.gnk

`web2.gnk` is a premium Gonka Name Service domain (.gnk).

It symbolizes the transition from the Web2 era (centralized platforms and siloed data stores) to decentralized AI computing
on the Gonka network.

## Why this is important

- The perfect selling point for Web2→Web3 / Web2→Gonka migration products
- A compelling, universally recognized term ("Web2") in a single, unique name, `.gnk`
- Can be used as:

- The primary entry point for a bridge/middleware project

- A landing page for Web2 migration to Gonka

- A brand anchor for decentralized AI marketing
